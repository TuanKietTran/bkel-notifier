# bkel-fetching

A simple program to transfer BK e-learning
messages to personal Telegram bot.

## How to use
1. Duplicate `.env.example` into `.env`.
2. Replace example keys with your personal ones.
3. ```shell
   go build .
   ```
4. ```shell
   ./bkel-fetching
   ```
5. (Optional) Set a crontab to run this program
periodically
